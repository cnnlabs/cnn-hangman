This is an app for playing hangman.

## Getting Started (The Docker Way)

```
$ docker-compose up
```

Navigate to localhost:8080 and start playing!

## Getting Started (The Non Docker Way)

```
$ npm install
$ nvm use
```

Terminal 1

```
$ npm start
```

Terminal 2

```
$ npm run start:fe:dev
```

Navigate to localhost:8080 and start playing!
